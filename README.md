# Cliente Inadiplente

###Depedencias 
- Instalado git, NoneJs, MongoDB, NPM

###Manual
- git clone https://github.com/tthiagopereira/cliente-inadiplente.git
###NodeJs - backend
- cd cliente-inadiplente
-  cd backend/
- npm install
- nodemon dist/main.js

###Vuejs - frontend
- cd cliente-inadiplente/
- cd frontend
- npm install
- npm run serve
